# 18. 4Sum

## Intuition

Sort the array first so that duplicate values become adjacent and can be skipped easily. Then, fix the first two elements and use the Two Pointer technique to find the remaining two elements whose sum equals the target.

## Approach

1. Sort the array.
2. Fix the first element using a loop.
3. Skip duplicate values for the first element.
4. Fix the second element using another loop.
5. Skip duplicate values for the second element.
6. Initialize two pointers:
   - `left = j + 1`
   - `right = nums.length - 1`
7. Calculate the sum of the four numbers.
8. If the sum equals the target:
   - Add the quadruplet to the answer.
   - Skip duplicate values.
   - Move both pointers.
9. If the sum is smaller than the target, move `left`.
10. If the sum is greater than the target, move `right`.
11. Repeat until all unique quadruplets are found.

## Complexity

- **Time Complexity:** O(n³)
- **Space Complexity:** O(1) *(excluding the output list)*
